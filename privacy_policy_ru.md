# Политика конфиденциальности приложения "MeteoStation"

**Дата вступления в силу:** 28.11.2025

## 1. Общие положения

Данное приложение "MeteoStation" (далее — "Приложение") предназначено для работы с метеостанцией на базе ESP32‑C3. Настоящая Политика конфиденциальности описывает, какие данные об устройстве и пользователе может обрабатывать Приложение, и как эти данные используются.

Используя Приложение, вы соглашаетесь с условиями этой Политики.

## 2. Обрабатываемые данные

Приложение **не запрашивает и не собирает** следующих данных:

- ФИО, адрес, электронная почта, номер телефона и иные персональные данные пользователя.
- Контакты, календари, СМС, история звонков.
- Фото, видео, файлы пользователя.
- Учётные записи, логины/пароли, идентификаторы аккаунтов.

Приложение **не требует регистрации** и не создаёт никаких учётных записей.

### 2.1. Данные устройства и подключения

Для работы с метеостанцией Приложение использует следующие технические данные:

- Информация о Bluetooth‑устройстве (имя устройства, MAC‑адрес, уровень сигнала) — только в рамках сканирования BLE для поиска метеостанции.
- Список Wi‑Fi сетей, доступных устройству пользователя — только для отображения списка сетей и выбора сети для метеостанции.
- IP‑адрес метеостанции (ESP32‑C3) — используется для подключения к встроенному веб‑серверу станции и получения метеоданных.

Эти данные используются **исключительно локально** на устройстве пользователя для работы функционала Приложения и **не передаются** разработчику или третьим лицам.

### 2.2. Данные метеостанции

Приложение отображает следующие данные, полученные от метеостанции:

- Температура
- Влажность
- Давление
- Высота (рассчитывается на основе давления)

Эти данные относятся к показаниям датчиков и **не являются персональными данными пользователя**.

## 3. Разрешения Android

Для корректной работы Приложение может запрашивать следующие разрешения:

- **Bluetooth / Bluetooth LE** — для поиска и подключения к метеостанции по BLE.
- **Доступ к местоположению (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION)** — требуется системой Android для сканирования BLE и Wi‑Fi сетей.
- **Доступ к состоянию сети и Wi‑Fi (ACCESS_NETWORK_STATE, ACCESS_WIFI_STATE, CHANGE_WIFI_STATE)** — для работы с Wi‑Fi подключением метеостанции.
- **Доступ в Интернет (INTERNET)** — для подключения к веб‑интерфейсу метеостанции по локальному IP‑адресу.

Разрешения используются **только для обеспечения работы основных функций** Приложения. Локация пользователя не сохраняется и не передаётся разработчику.

## 4. Хранение и передача данных

- Приложение **не отправляет** данные на внешние серверы разработчика.
- Приложение **не использует сторонние аналитические сервисы** (Google Analytics, Firebase Analytics и т.п.).
- Приложение **не содержит рекламных SDK** (AdMob и т.п.) и не передаёт данные рекламным сетям.
- Вся обработка метеоданных и параметров подключения выполняется локально на устройстве пользователя и на метеостанции (ESP32‑C3).

## 5. Доступ третьих лиц

Разработчик Приложения не передаёт и не продаёт какие‑либо данные третьим лицам. 

Приложение не интегрировано с социальными сетями и не использует механизмы обмена данными с другими приложениями, за исключением стандартного сетевого взаимодействия с метеостанцией по BLE и HTTP.

## 6. Дети

Приложение предназначено для использования взрослыми пользователями и технически подкованными энтузиастами. Приложение не ориентировано специально на детей до 13 лет и не собирает осознанно персональные данные детей.

Если вы являетесь родителем или опекуном и считаете, что ваш ребёнок предоставил какие‑либо данные через Приложение, свяжитесь с разработчиком для их удаления (при наличии таких данных).

## 7. Права пользователя

Так как Приложение не собирает и не хранит персональные данные на стороне разработчика, пользователю не требуется подавать отдельные запросы на удаление или предоставление своих данных.

Вы можете в любой момент:

- Удалить Приложение с устройства, 
- Очистить данные приложения через настройки Android.

Это полностью удалит все локальные настройки и кэш, связанные с Приложением.

## 8. Изменения в Политике конфиденциальности

Разработчик оставляет за собой право вносить изменения в данную Политику конфиденциальности. Актуальная версия всегда доступна по указанному URL.

При существенных изменениях может отображаться уведомление внутри Приложения или в описании на Google Play.
